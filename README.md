```
### ­ЪЊї рЃЊрЃћрЃарЃўрЃЋрЃљрЃфрЃўрЃљ: `int x1 = 42;`
```

РЄњ <type> <id> = <num> ;

­ЪњА рЃљрЃ«рЃАрЃюрЃљ:

рЃЏрЃћрЃЮрЃарЃћ рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃЋрЃљ: рЃЊрЃћрЃЎрЃџрЃљрЃарЃљрЃфрЃўрЃљ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃўрЃЌ (int x = 5;)

­ЪЊЮ BNF #4: While рЃфрЃўрЃЎрЃџрЃў / рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ

рЃљрЃдрЃгрЃћрЃарЃћрЃЌ BNF рЃцрЃЮрЃарЃЏрЃўрЃЌ рЃњрЃарЃљрЃЏрЃљрЃбрЃўрЃЎрЃљ while рЃфрЃўрЃЎрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф:

рЃърЃўрЃарЃЮрЃЉрЃљ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃерЃў: рЃфрЃЋрЃџрЃљрЃЊрЃў, рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў (<, >, =), рЃфрЃЋрЃџрЃљрЃЊрЃў

рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ: рЃфрЃЋрЃџрЃљрЃЊрЃў = рЃарЃўрЃфрЃ«рЃЋрЃў

```

### ­ЪЊї рЃЊрЃћрЃарЃўрЃЋрЃљрЃфрЃўрЃљ: `while (x < y) do z = 5`
```

РЄњ while ( x < <var> ) do <assign>
РЄњ while ( x < y ) do <assign>
РЄњ while ( x < y ) do <var> = <num>

РЄњ while ( x < y ) do z = 5
­ЪњА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃЎрЃЮрЃЊрЃў: while (x=y) do {5/0}
рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў:
